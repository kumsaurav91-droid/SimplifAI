FAQ
Aksar Pooche Gaye Sawaal ❓
Yahan sab clear ho jaayega — promise!
SimplifAI aur ChatGPT mein kya fark hai?
▼
ChatGPT ek general-purpose assistant hai — woh English mein, technical language mein explain karta hai. SimplifAI specifically Indian students ke liye banaya gaya hai. Yeh Hinglish mein samjhata hai, Indian analogies use karta hai (cricket, biryani, IPL!), aur sirf code samjhane par focus karta hai. Saath mein milta hai animated Complexity Visualizer aur 5-question Quiz — jo ChatGPT bilkul nahi deta. Ek tool, complete learning experience!
Kya yeh beginners ke liye bhi useful hai?
▼
Bilkul! SimplifAI 3 levels mein kaam karta hai: Beginner mein dum mein ekdum simple language — koi jargon nahi, bas relatable examples. Intermediate mein patterns aur logic flow samjhata hai. Advanced mein deep technical insights, edge cases, aur best practices. Tum apna level choose karo — beginner ho ya experienced developer, dono ke liye useful hai!
Konsi programming languages support hoti hain?
▼
Dedicated support hai Python, JavaScript, Java, C++, aur SQL ke liye — badges se select karo. Lekin "Auto Detect" mode mein practically koi bhi language kaam karti hai — TypeScript, Go, Rust, PHP, Swift, Kotlin sab. Amazon Nova 2 Lite ke through almost har popular language explain ho sakti hai!
Mera code safe hai? Kahin store toh nahi hota?
▼
Haan, tumhara code 100% safe hai. Jo code tum paste karte ho woh sirf explanation generate karne ke liye Amazon Nova 2 Lite ko bheja jaata hai — hum koi bhi code apne servers pe store nahi karte. Browser tab band karo, data gone. Koi account required nahi, koi login nahi, koi data collection nahi. Sensitive ya production code paste karna avoid karo — yeh tool sirf learning ke liye hai!
PDF Report mein kya kya hota hai?
▼
PDF Report ek professional 4-page document hai jo automatically generate hoti hai. Ismein hota hai: cover page (language, complexity, quiz score), submitted code with line numbers, executive summary, time & space complexity analysis with Big-O scale, optimization suggestions, aur complete quiz results with per-question breakdown. Apne professor ko dikhao, portfolio mein add karo, ya apni progress track karo — download karo, share karo!
Quiz kaise kaam karta hai? Questions kahan se aate hain?
▼
Quiz tumhare specific code ke basis pe generate hota hai — generic questions nahi! Agar tumhare code mein nested loops hain toh O(n²) ka question milega. Functions hain toh reusability ka. Arrays hain toh data structures ka. 5 MCQ questions milte hain, har answer ke baad instant feedback milta hai Hinglish mein with analogy, aur end mein Grade (S/A/B/C) aur detailed breakdown. Phir quiz PDF mein bhi aa jaata hai!
